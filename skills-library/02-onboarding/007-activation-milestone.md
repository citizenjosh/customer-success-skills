## STRATEGIC OBJECTIVE

Establish a confirmed behavioral anchor for measuring onboarding success.

## DATA INPUTS

- **Product** telemetry
- **Historical** trial-to-paid data

## EXECUTION LOGIC

1. **Analyze** historical data to identify the specific behavior that predicts long-term retention.
2. **Define** this behavior as the "**Activation Event**" for the customer segment.
3. **Monitor** real-time telemetry to track the percentage of users reaching this milestone.

## GUARDRAILS

- **REQUIRE** the "**Activation Event**" to be a behavior that is consistently linked to long-term retention.
- **ENSURE** activation data is updated in real-time to allow for immediate intervention.
- **ENSURE** the Activation Event represents a Value Realization action (e.g., first report) rather than a session count.
