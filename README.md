## The Vision: Increase company profit by scaling the Customer Success Manager via agentic operations.

This is a sovereign library of specialized AI Skills designed to reduce the toil in Customer Success operations. These skills are written to be read by humans and executed by autonomous agents (Gemini, Claude, etc.).

## Key Pillars of the Framework
1. **Technical Integrity:** Moving beyond basic CRM notes to multi-protocol schema alignment (gRPC, OpenAPI).
2. **Outcome Intent:** Codifying exactly what the customer wants to achieve from Day 0.
3. **Value Receipts:** Automating the evidence of ROI for CFO-level reviews.
4. **Predictive Save-Plays:** Using neural telemetry to identify and stop churn 90 days in advance.

## How to Use This Framework

### 1. The Skill Library (The Engine)
The core of this repository is the `/skills-library`. These are raw, agent-ready files meant to be ingested by AI Orchestrators or read by humans.

### 2. The Web Portal (The Dashboard)
We have included a React-based portal to make these skills accessible to non-technical users. 
* **Setup Required:** You must deploy this repository to a hosting provider (like Vercel or Netlify) to make the portal live. 
* **For CSMs:** Once live, navigate to your custom URL to browse playbooks and guardrails.

### 3. Agentic Integration (The Automation)
Point your AI agents (via MCP or API) to the `/skills-library` to allow them to follow the Execution Logic autonomously.
