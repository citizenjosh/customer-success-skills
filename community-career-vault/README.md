Welcome to the **Community Career Vault**. This space is a dedicated repository for open-source AI skills, agent prompts, and blueprints focused on personal career acceleration, career transitions, and job search optimization for Customer Success, Account Management, and RevOps professionals.

While the core `skills-library/` maps to corporate, post-sale operations, this vault is built specifically by the community, for the community, to help practitioners navigate organizational changes, layoffs, and job hunts with high-leverage AI tools.

## What Belongs Here
* **Resume & LinkedIn Optimization:** Frameworks and prompts to help translate tactical day-to-day CS actions into high-impact, business-oriented revenue achievements.
* **Interview Simulation & Strategy:** Logic gates and behavioral prompt sequences tailored for competitive SaaS post-sales and operations roles.
* **Outbound Search Automation:** Playbooks for organizing, personalizing, and managing an active outbound job search pipeline cleanly and authentically.

## Structural Scope & Architecture
This directory intentionally sits completely separate from the primary `skills-library/` operational data pipeline. 

Contributions placed inside this directory **bypass** the automated system vetting scripts (`scripts/vetting_agent.py`). Because these are tools for personal use rather than live software integrations, skills in this vault do not require Model Context Protocol (MCP) data configurations, CRM connections, or database schemas. 

They are designed to be copied, customized, and run safely in any local LLM interface or personal AI environment.

## Contributing a Career Skill
If you have engineered a prompt setup or career framework that helped you or your peers land an opportunity:
1. Create a markdown file inside this directory describing the workflow (e.g., `linkedin-profile-transformer.md`).
2. Include your name and your LinkedIn or GitHub profile at the top of the file so you receive permanent credit as a contributor.
3. Open a Pull Request targeting this directory.
